# Mahathi Jammi Streamlit App

This repository contains a simple Streamlit app with the title set to **Mahathi Jammi**.

## Run the app

```bash
streamlit run app.py
```

## Requirements

- Python 3.8+
- Streamlit

Install Streamlit with:

```bash
pip install streamlit
```

## Description

The app displays your name as the title and includes a text input, a message field, and a greeting button. It also shows a live summary of what you enter.
